# cumu

A self-hosted music streaming web application built with Node.js.

## Features

- [+] First-time setup wizard
- [+] Admin panel with album/song management
- [+] MP3 metadata auto-extraction
- [+] Album cover artwork support
- [+] Playlist creation and management
- [+] Audiobook/Hörspiel mode with 15s skip
- [+] Responsive design (mobile + desktop)
- [+] User management (Admin + Creator roles)
- [+] Search across songs, albums, artists, playlists
- [+] Play history and recommendations

## Setup

```bash
npm install
node src/server.js
```

Open `http://localhost:3000` in your browser and follow the setup wizard.

## Requirements

- Node.js >= 18
- ~50MB disk space for the app (music stored separately)
